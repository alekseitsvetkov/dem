<threat_model>
## Security Threat Model

**Scope:** Local CLI skeleton, command registration, and JSON output helpers.

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Success output accidentally mixes diagnostics into stdout, breaking scripts that parse JSON. | medium | Reserve stdout for success JSON only; errors and diagnostics go to stderr through Plan 02's centralized error path. |
| Command handlers call `os.Exit`, making future tests and embedding brittle. | medium | Keep process exit behavior in `cmd/dem/main.go`; command handlers return errors. |
| Future commands bypass the JSON envelope and emit ad hoc output. | medium | Add shared `internal/output.Response` and `output.WriteJSON` helper in this plan. |
</threat_model>
