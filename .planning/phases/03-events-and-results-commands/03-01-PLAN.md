## Security Threat Model

**Scope:** Domain models, parser error taxonomy, HTML parser fixtures, parser implementations.

| Threat | STRIDE | Severity | Disposition | Mitigation |
|--------|--------|----------|-------------|------------|
| Parser tests accidentally fetch live HLTV pages | Spoofing | High | Mitigate | Tests must only read from `internal/hltv/parser/testdata/`; no `net/http` imports in test files |
| Parser errors leak raw HTML content | Information Disclosure | Medium | Mitigate | `Details()` returns only code, area, field, message — never includes raw HTML or response bodies |
| Tier extraction silently fails (no `.event-tier` element found) | Tampering | Low | Accept | Tier defaults to empty string; no crash, no error — downstream provider filter becomes a no-op for tier="" |
| Parser selector assumptions diverge from live HLTV markup | Spoofing | Medium | Accept | Fixture-tested parsers; selector adjustments use fixture updates as the change mechanism |
</threat_model>

       ```go
       func ParseEvents(r io.Reader, sourceURL string) ([]domain.Event, error) {
           doc, err := goquery.NewDocumentFromReader(r)
           if err != nil {
               return nil, &ParseError{Code: ErrorCodeParse, Area: "events", Err: err}
           }

           var events []domain.Event
           doc.Find(".event").Each(func(i int, s *goquery.Selection) {
               id, exists := s.Attr("data-event-id")
               // validate required: id and name
               // if missing required field, set events=nil, err=ParseError{...}, return false (break Each)
               // ...
               evt := domain.Event{
                   ID:        id,
                   Name:      nameText,   // from link with class .event-name
                   StartDate: startDate,  // from .event-date, split on " to "
                   EndDate:   endDate,
                   Location:  location,   // from .event-location text
                   Tier:      tierText,   // from .event-tier text, trimmed
                   SourceURL: resolveURL, // resolve relative event link href, fallback to sourceURL
               }
               events = append(events, evt)
           })
           return events, err
       }
       ```

       Key implementation details:
       - Required fields: event ID (`data-event-id` attr on `.event`) and name (text of `.event-name` link). If either is missing/empty, return `*ParseError{Code: ErrorCodeParse, Area: "events", Field: "id"}` or `Field: "name"`.
